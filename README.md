# Basic Marimo

Памятка по созданию этого репозитория:
уже должен быть установлени `uv manager`.
Инициализируем
```
uv init
```
устанавливаем `marimo`:
```
uv add marimo
```
наслаждаемся результатом:
```
uv run marimo edit
```
в браузере должен открыться редактор блокнотов.
Можно приступать к работе
