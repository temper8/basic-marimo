# Basic Marimo

Памятка по созданию этого репозитория:
уже должен быть установлени `uv manager`.
Инициализируем
```
uv init
```
устанавливаем `marimo`:
```
uv add marimo
```
наслаждаемся результатом:
```
uv run marimo edit
```
в браузере должен открыться редактор блокнотов.
Можно приступать к работе

блокноты:

- note_01 - Построение графика с MatPlotLib
- note_02 - Aнимация с помощью slider
- note_03 - Рисуем множество Мандельборта
- note_04 - Aнимация Мандельброта
